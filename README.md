Title:
Simple C++ Calculator Program

Description:

This C++ program performs basic arithmetic operations: addition, subtraction, multiplication, and division.
The user inputs two numbers and an operator (+, -, *, /). Based on the selected operator, the program calculates and displays the result.
It also includes error handling for division by zero and invalid operators.

How to Run (Run Instructions):
1.Save the Code

Save the program in a file, for example:
calculator.cpp

2️.Compile the Program

Open your terminal or command prompt and run:
g++ calculator.cpp -o calculator

3️. Run the Program

After successful compilation, run:
./calculator

4️. Input Steps

Enter the first number
Enter an operator (+, -, *, /)
Enter the second number

The program will display the result.
